# Othello Orbit

A polished browser-based Othello/Reversi game built as a lightweight static web app.

## Features

- Live legal-move highlighting with flip counts
- Score tracking and last-move emphasis
- Move history with undo support
- Strategic hint button
- Theme toggle with local preference persistence
- Zero-build local run: open `index.html` or serve the folder

## Verification

```bash
node --test
python3 -m http.server 8000
```
